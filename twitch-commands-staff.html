🛡️ Arsenal du Modérateur
// Modération Chat
⚠️
!warn [pseudo] MODO
Avertit un utilisateur. Le compteur s'affiche (ex : 2/3). Au dépassement du seuil : timeout automatique progressif 60s → 300s → 600s.
⏱️
!timeout [pseudo] [durée] MODO
Timeout direct pour la durée spécifiée (défaut 60s). Confirmation affichée dans le chat.
🧹
!clearwarn [pseudo] MODO
Efface tous les avertissements accumulés d'un utilisateur. Remet le compteur à zéro.
// File de Modération AutoMod
📋
!queue MODO
Affiche les 3 premiers messages en attente de validation : ID, auteur, contenu et raison de mise en file.
✅
!approve [id] MODO
Approuve un message en attente. Le message est libéré dans le chat avec confirmation.
❌
!reject [id] MODO
Rejette et supprime définitivement un message en attente. L'utilisateur est notifié selon la config AutoMod.
// Overlays & Compteurs Stream
🎯
!kill [n] MODO
+N kills au compteur overlay. Mise à jour WebSocket temps réel dans OBS/Streamlabs.
💀
!die [n] MODO
+N morts au compteur overlay. Déclenche optionnellement son + animation GIF configurés.
📦
!extract [n] MODO
+N extractions au compteur overlay. Animation en temps réel.
🔁
!raz [compteur] MODO
Remet à zéro un compteur : extractions | morts | kills. Notification overlay instantanée.
Usage : !raz extractions
// Arc Raiders — Compteurs Dédiés
⚔️
!arc_kill · !arc_die · !arc_pvp · !arc_extract MODO
Compteurs Arc Raiders dédiés. Kills, morts PvE, morts PvP, extractions — chacun tracké séparément dans l'overlay.
🔄
!arc_reset MODO
Réinitialise tous les compteurs Arc Raiders à zéro pour une nouvelle session.
// Chat & Annonces
📢

🔔
!alert [texte] MODO

📡

Shoutout pour un autre streamer. Récupère le jeu actuel + uptime via API Twitch Helix. Génère une blague IA sur le jeu.
